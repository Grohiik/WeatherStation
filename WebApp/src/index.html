About the Hugin project

Before the idea of Hugin had touched our minds, our project started out as a Weather Station system called Poseidon. Both Hugin and Poseidon are made up of the same fundamental idea, namely, an embedded system which collects data that in turn is sent to a server. The server formats and sorts this data into something comprehensible which will then be utilised by a website and be presented to the user whom has purchased the system.
